# Configuration

## 1. Install dependencies

```
yarn install
```

## 3. Run the app

```
yarn dev
```

# Technologies

- React
- TypeScript
- Material UI (for UI components)
- TanStack React Query (for data fetching)
- dexie.js (for IndexedDB)

# Deployment

The app is deployed on Vercel. You can access it here: https://project-starboard.vercel.app/
